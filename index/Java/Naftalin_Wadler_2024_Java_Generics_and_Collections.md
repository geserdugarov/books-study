# Java Generics and Collections (Second Edition)

## Book Metadata
- **Title:** Java Generics and Collections: Fundamentals and Recommended Practices
- **Authors:** Maurice Naftalin, Philip Wadler
- **Publisher:** O'Reilly Media
- **Year:** 2024 (Second Edition, Early Release)
- **Pages:** ~300 (estimated, early release)
- **ISBN:** 978-1-098-13666-6

## Target Audience and Prerequisites
- Java developers wanting to master generics and the Collections Framework
- Programmers who use generics but want to understand the theory behind wildcards, bounds, and type erasure
- API designers needing to create flexible, type-safe generic interfaces
- Requires solid Java fundamentals; some familiarity with basic generics helpful

## Overall Focus
The canonical deep dive into Java generics and the Collections Framework, co-authored by Philip Wadler who co-designed the generics specification. This second edition updates the classic 2006 text with modern Java features. Part 1 covers generics theory — subtyping, wildcards, bounds, type erasure, reification — while Part 2 covers the Collections Framework in detail. The book provides the theoretical foundation for understanding why Java's type system works the way it does.
